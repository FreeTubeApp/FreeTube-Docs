# Arabic documentation (`ar/`)

This directory contains Arabic translations of FreeTube documentation pages.

- URLs are prefixed with `/ar/` (for example `/ar/faq/`).
- Pages not yet translated still link to the English documentation where noted.
- UI strings in the FreeTube app are translated separately via [Weblate](https://hosted.weblate.org/engage/free-tube/).

To add or improve translations, edit the Markdown files here and open a pull request on [FreeTube-Docs](https://github.com/FreeTubeApp/FreeTube-Docs).
